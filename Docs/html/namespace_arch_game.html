class  LoadableSet
 The LoadableSet is a class that keeps 2 lists of IArchLoadables to be used in the content loading phase. The preloadables will be loaded before the window appears and is intended for content critical for the game at any given moment. The loadables will be loaded while the loading state is being shown on screen, in a background thread. It is reccommended to load all the content of the game in one of those lists, and to have the preloadables as few as possible. See the wiki for more information. More...
