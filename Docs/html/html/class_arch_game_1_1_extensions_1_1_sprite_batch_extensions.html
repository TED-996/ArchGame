Draws a Texture2D using a special algorithm, that fills the destinationRectangle with the texture. The borders of the destinationRectangle will be filled with the borders and corners of the source texture. The center of the destination rectangle will be filled with the center of the source texture. Absolutely no stretching will occur, only tiling.

Parameters
spriteBatchThe SpriteBatch to draw with.
textureThe texture to use
destinationRectangleThe rectangle to draw on
sourceRectangleNThe source rectangle to draw. May be null.
colorThe color of the tint to use. Use Color.White for no tint.
borderPercentageThe percentage of the source rectangle (or border bounds) to consider the border.
